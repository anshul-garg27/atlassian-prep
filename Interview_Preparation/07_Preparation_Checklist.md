### **WEEK 5: BEHAVIORAL PREP**

**Focus:** Atlassian Values + STAR Stories

#### ‚úÖ **Day 1-2: Values Study**
- [ ] Read [Atlassian Values Guide](https://www.atlassian.com/company/values)
- [ ] Watch Atlassian culture videos
- [ ] Understand what each value means in practice

#### ‚úÖ **Day 3-5: Story Preparation**
Prepare 3 stories for EACH value (15 total):

**Template for Each Story:**
```markdown
## Story: [Short Title]
**Value:** [Which value this demonstrates]
**Situation:**
- Context: [Company, team, timeline]
- Challenge: [What was the problem]

**Task:**
- Your role: [Your responsibility]
- Goal: [What needed to be achieved]

**Action:**
- Step 1: [What you did]
- Step 2: [Next action]
- Step 3: [And so on...]

**Result:**
- Outcome: [What happened]
- Metrics: [Quantifiable impact]
- Learning: [What you learned]
```

- [ ] Write out 15 full stories
- [ ] Each story should be 2-3 minutes when spoken
- [ ] Include specific names, dates, metrics

#### ‚úÖ **Day 6-7: Practice**
- [ ] Practice with friend/mock interviewer
- [ ] Record yourself and listen back
- [ ] Time yourself (should be ~2.5 min per story)
